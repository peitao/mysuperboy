# Hello Skill

## Overview

This is a demo skill that provides greeting and time utilities.

## Tools

- `hello`: Say hello to someone (参数: name)
- `current_time`: Get current time
- `echo`: Echo back text

## Usage

```
Use hello tool to greet: hello(name="Tom")
Use current_time to get time
Use echo to repeat text: echo(text="hello")
```
